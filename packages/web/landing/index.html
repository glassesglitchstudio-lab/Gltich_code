22 Bug Düzeltmesi — 14 Dosya, 268 Değişiklik

Kritik güvenlik, performans ve stabilite düzeltmeleri. 6 kritik, 8 orta, 8 düşük seviye.

🔴Postinstall script — npm kurulumu artık çalışıyor (postinstall.mjs)
🔴Response double-read fix — response.json() artık sadece bir kez okunuyor (webfetch.ts)
🔴SSRF koruması — private IP adreslerine erişim engellendi (webfetch.ts)
🔴Fix pipeline timeout/retry — 3 deneme hakkı, exponential backoff (fix/index.ts)
🔴Dosya yazma hatası — fix dosyaları artık diske yazılıyor (fix/index.ts)
🟡XSS koruması — session.title escape edildi (plus-two-coder.ts)
🟡DB resource leak — db.close() finally bloğuna taşındı (db.ts)
🟡Graceful shutdown — web.ts sunucusu artık düzgün kapanıyor
🟡PR collision fix — remote'da aynı isimde PR varsa uyarı (pr.ts)
🟡Solve limitleri — max-steps: 50, max-parallel: 3 (solve/index.ts)
🟢CSV injection — benchmark title'daki özel karakterler escape ediliyor
🟢Stderr memory leak — 1MB ile sınırlı (voice.ts)
🟢Import hata mesajı — dosya bulunamama vs JSON parse ayrıştırıldı (import.ts)
🟢Terminal cleanup — setRawMode TTY kontrolü ile sarmalandı (terminal.ts)

Nasıl Geliştirildi?

v0.2.35'in geliştirme süreci: analiz, düzeltme, test ve publish.

🔍

1. Otomatik Kod Analizi

self-supervision tool'u ile tüm codebase tarandı. 48 potansiyel bug tespit edildi. Critical/orta/düşük seviyelere göre sıralandı.

🤖

2. AI-Coded Düzeltmeler

Her bug için self-healing loop çalıştı: kök neden analizi → fix üretimi → uygulama. 22 bug tek oturumda düzeltildi.

🧪

18 paket için tsgo --noEmit çalıştırıldı. TypeScript hataları sıfırlandı. Lint temiz.

📦

4. Git & CI/CD

git add → git commit → git push. GitHub Actions otomatik tetiklendi. 13 platform build başladı.

🏷️

5. Version Tag & Publish

git tag v0.2.35 ile versiyon sabitlendi. publish.yml npm'e publish etti.

📊

6. İstatistikler

14 dosya, 268 ekleme, 76 silme. 18 dosya değiştirildi. Toplam süre: ~10 dakika.
